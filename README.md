# Simulador de Presupuestos y Control de Stock

## Integrantes
Morales leandro,Nuñez gerardo,Rodriguez Nicolas

## Descripción

Proyecto realizado en Node.js utilizando ES Modules lo cual nos permite:

Gestionar un inventario de productos
Simular una venta
Calcular subtotal
Aplicar descuentos
Calcular IVA
Mostrar el total
Actualizar el stock
